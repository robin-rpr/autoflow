Autoflow is a Workflow Automation Tool.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Install

Install dependencies:

```bash
npm install
```

### Run

Start the development server:

```bash
npm run dev
```

Open your browser to `http://localhost:5173`
